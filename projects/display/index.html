Pantalla medidora

Un medidor de temperatura, humedad y luminosidad en tiempo real, además de incorporar un cronómetro con alarma, todo controlado mediante un mando de infrarrojos. Para su implementación se utilizó un Arduino UNO R3, usando C++ para programarlo.

Se compone de un display 16x2, un buzzer pasivo, un sensor infrarrojo, un potenciómetro, una fotorresistencia, y un sensor de temperatura y humedad.
